# uttarakhand-homestay-dataset-generator
Python project to generate a synthetic Uttarakhand homestay and eco-tourism guest review dataset for BI, SQL, and NLP projects. It creates realistic review data with ratings, text, location, and metadata for sentiment analysis and dashboard development.


OUTPUT Columns 

Column	Purpose
review_id	Primary Key
homestay_id	Foreign Key
homestay_name	Property
district	Geographic Analysis
state	State
guest_type	Segmentation
rating	KPI
review_text	NLP
sentiment	Target
review_topic	Theme Analysis
review_date	Trend Analysis
host_response	Service KPI
stay_duration_days	Guest Behavior
booking_channel	Source Analysis


SQL Schema

homestays
reviews
sentiment_analysis
